03 / SRI Architecture

SOVEREIGN

ROUTING.

Semantic Routing Intelligence understands your query and dispatches it to the best model — local first, always. When cloud is needed, every model is containerised to a specific folder. The private AI sanitises memories and stores them in your vault.

01
Sandboxed Cloud

Cloud models see only their assigned folder. No access to your full vault. Ever.

02
Memory Sanitisation

Private AI cleanses cloud responses before storing locally. No raw cloud memory persists.

03
TOR Toggle

Cloud calls route through TOR optionally. Your IP never reaches the model provider.
