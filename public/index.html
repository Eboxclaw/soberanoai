03 / SRI Architecture

SEMANTIC

ROUTING.

SRI understands your query and dispatches to the best model — local first, always. Cloud models are containerised to a specific folder. The private AI sanitises memories before writing to your vault.

01
Local First

Ollama or on-device model handles 94% of queries instantly. No network call, no latency, no exposure.

02
Exo Mesh Fallback

If local RAM is insufficient, SRI distributes inference across your Tailscale-connected devices via Exo.

03
Sandboxed Cloud

Cloud models see only their assigned folder. PII scrubbed. TOR optional. Memory quarantined before vault entry.
